1. Как узнать длину строки?

string text = "Hello";
int length = text.Length; // 5

2. Как получить предпоследний символ строки?

char secondLast = text[text.Length - 2];

3. Что такое конкатенация строк?

1) Оператор +

string result = s1 + " " + s2;

2) Интерполяция

string greeting = $"Привет, {name}!";

3) Метод string.Concat

string result = string.Concat(s1, s2);

4) Метод string.Join (для объединения массива строк с разделителем):

string[] words = {"Яблоко", "Банан", "Груша"};
string result = string.Join(", ", words); // "Яблоко, Банан, Груша"

5) StringBuilder (для создания строк в циклах):

System.Text.StringBuilder sb = new System.Text.StringBuilder();
for (int i = 0; i < 5; i++)
{
	sb.Append("Строка ");
	sb.Append(i);
	sb.AppendLine(" - добавляем текст.");
}
string finalString = sb.ToString();
