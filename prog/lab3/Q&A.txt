1. Как узнать длину строки?

string text = "Hello";
int length = text.Length; // 5

2. Как получить предпоследний символ строки?

char secondLast = text[text.Length - 2];

3. Что такое конкатенация строк?

1) Оператор +

string result = s1 + " " + s2;

2) Интерполяция

string greeting = $"Привет, {name}!";

3) Метод string.Concat

string result = string.Concat(s1, s2);

4) Метод string.Join (для объединения массива строк с разделителем):

string[] words = {"Яблоко", "Банан", "Груша"};
string result = string.Join(", ", words); // "Яблоко, Банан, Груша"

5) StringBuilder (для создания строк в циклах):

System.Text.StringBuilder sb = new System.Text.StringBuilder();
for (int i = 0; i < 5; i++)
{
	sb.Append("Строка ");
	sb.Append(i);
	sb.AppendLine(" - добавляем текст.");
}
string finalString = sb.ToString();

4. Строка – это неизменяемый объект. Как тогда осуществляется замена  
подстроки методом Replace?

Символы: str.Replace('a', 'b') — меняет все символы 'a' на 'b'.

Строки: str.Replace("old", "new") — меняет все вхождения подстроки "old" на "new".

Цепочка замен: str.Replace("a", "b").Replace("c", "d") — можно объединять несколько замен в одну строку.

string original = "Hello World";
string replaced = original.Replace("World", "C#");

Если подстрока не найдена, Replace возвращает исходную строку без изменений.

6. Какие методы существуют для сравнения строк?

String.Compare - сравнивает значения двух строк. Возвращает целочисленное значение.

String.CompareOrdinal -сравнивает две строки без учета локального языка и региональных параметров. Возвращает целочисленное значение.

String.CompareTo - сравнивает текущий объект строки с другой строкой. Возвращает целочисленное значение.

String.StartsWith  - определяет, начинается ли строка с переданной строки. Возвращает логическое значение.

String.EndsWith - определяет, заканчивается ли строка переданной строкой. Возвращает логическое значе

String.Contains - определяет, находится ли символ или строка в другой строке. Возвращает логическое значение.

String.Equals - определяет, совпадают ли две строки. Возвращает логическое значение.

String.IndexOf  - возвращает позицию индекса символа или строки, начиная с начала проверяемой строки. Возвращает целочисленное значение.

String.LastIndexOf  - возвращает позицию индекса символа или строки, начиная с конца проверяемой строки. Возвращает целочисленное значение.

7. Что такое «интерполяция строк»?

Интерполяция строк — встраивание выражений в строку через $:

int x = 5; string s = $"x = {x}";

8. Как убрать лишние пробелы в начале и в конце строки?

Для этого используется метод Trim(). Он удаляет все «невидимые» символы (пробелы, табы, переносы строк) с обоих кусков строки.

Если нужно убрать пробелы только в начале — используйте TrimStart().

Если только в конце — TrimEnd().

string text = "  hello world   ";
string cleanText = text.Trim(); // Результат: "hello world"

9. Что делает со строкой метод Split?

Метод Split разделяет одну строку на массив подстрок, основываясь на заданном разделителе (символе или строке). Это очень полезно, когда нужно «разобрать» предложение на отдельные слова или данные из CSV-файла.

string data = "apple,banana,orange";
string[] fruits = data.Split(','); 
// Результат: массив ["apple", "banana", "orange"]

10. Как используется метод String.Format?

Метод String.Format позволяет вставлять значения переменных в шаблон строки. В шаблоне используются фигурные скобки с индексами {0}, {1} и так далее, которые заменяются на соответствующие аргументы.

string name = "Alice";
int age = 20;

// Шаблон: "Name: {0}, Age: {1}"
string result = string.Format("Name: {0}, Age: {1}", name, age);
// Результат: "Name: Alice, Age: 20"

11. Как перевести всю строку в верхний/нижний регистр?

Для этого используются методы ToUpper() и ToLower(). Они создают новую строку, где все буквы заменены на соответствующие регистры.

string original = "Hello C#";

string upper = original.ToUpper(); // "HELLO C#"
string lower = original.ToLower(); // "hello c#"

12. Как получить объект DateTime с требуемой датой?

Самый простой способ — использовать конструктор класса DateTime, передав в него год, месяц и день (в таком порядке).

// Создаем объект на 25 декабря 2024 года
DateTime myDate = new DateTime(2024, 12, 25);

// Можно добавить время (часы, минуты, секунды)
DateTime myDateTime = new DateTime(2024, 12, 25, 14, 30, 0);
Также можно использовать DateTime.Parse("25.12.2024"), но конструктор надежнее, так как он не зависит от настроек языка в системе.

13. Как получить день недели, который будет через 20 дней после сегодняшней даты?

Для этого нужно использовать свойство DateTime.Today (или Now), применить метод .AddDays(20), а затем обратиться к свойству DayOfWeek.

DateTime today = DateTime.Today;
DateTime futureDate = today.AddDays(20);
DayOfWeek day = futureDate.DayOfWeek;

Console.WriteLine(day);

14. Для чего используется тип данных TimeSpan?

TimeSpan используется для представления интервала времени или длительности (разницы между двумя датами). В отличие от DateTime, который указывает на конкретный момент на временной шкале, TimeSpan — это просто количество времени (сколько дней, часов или минут прошло).

DateTime d1 = inputDate;
DateTime d2 = DateTime.Today;

TimeSpan difference = d1 - d2; 

int days = difference.Days; 
return Math.Abs(days);
